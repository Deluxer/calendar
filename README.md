# Calendar

Calendar to schedule events or task

## instalation

```
yarn install
```
<br />
Build to create */dist* folder
<br />

```
yarn build
```
<br />

Run in dev
<br />
```
yarn dev
```
<br />

Run in prod
<br />
```
yarn preview | npm run preview
```


## Dependencies

- Browser Route
- Bootstrap 5
- Font aewsome (cdnjs)
- react-big-calendar
- react-modal
- sweetalert2
- redux toolkit

## Stack technologies
* React with redux toolkit and typescript
